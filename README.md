# michis-mod-bot

This is a youtube livestream moderation bot specifically designed for "Der Michi".

What to install:

-   python 3.8.5
-   pip
    -   googleapiclient
    -   google_auth_oauthlib

Make a Youtube account for the bot, give it mod rights.
At start choose to log in via its account.
